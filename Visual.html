Visualisation 3D de monuments

Application 3D premettant la visualisation du Rocher St Michel D'Aihuilhe

Projet tutoré de 1ère année. Réalisé sur Unreal Engine 4. Projet de groupe composé de 5 personnes. Il fallait créer une application permettant de visualiser le Rocher St Michel d' Aiguilhe. Pour ce faire, nous avions un modèle scanné en 3D du rocher fourni par la société 3D Process. Nous avons créé un système de visualisation avec des fonctions de rotation et de zoom. Aussi des points d'intéractions qui une fois cliqué faisait apparaître différentes informations sur le rocher. Il est aussi possible de visualiser le rocher de nuit.

Une partie de la ville a était recrée de manière abstraite pour pouvoir mieux intégrer le modèle sans pour autant le masquer. Ce projet est terminé
