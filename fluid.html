Simulation de fluides | Fluid Simulation

Simulation de fluides avec FleX de Nvidia sur Unity | Fluid Simulation with NVIDIA Flex on Unity

Projet tutoré de licence professionelle, en tant que chef de projet, modélisateur et programmeur. Réalisé sur Unity. Ce projet permet la visualisation en temps réel d'un flux de sang circulant dans une aorte. Pour les besoins du projet il est possible de modifier les propriétés du liquide mais aussi de le visualiser d'une autre manière.

This was project was realized as Project Manager, programmer and modeler. Created on Unity. This project allows in real time the visualisation of a blood flow in an aorta. Because of the needs of the project, it's possible to modify the liquid's properties and to have different ways of visualisation.

NVIDIA FleX

FleX est une librairie open source mis à disposition par NVIDIA elle permet la simulation de fluides mais aussi de vêtemement, de corps déformables ou solides

FleX is an open source library provided by NVIDIA. It allows fluid simulation, but also clothes, deformables or solids bodies
