Création d'une Application 3D temps réel avec Unreal Engine 4

Projet tutoré de 1ème année. Réalisé sur Unreal Engine 4. Projet de groupe composé de 5 personnes. Il fallait creer une application permettant de visualiser le Rocher St Michel d' Aiguilhe. Pour ce faire, nous avions un modèle scanné en 3D du rocher fournit par la société 3D Process. Nous avons créé un système de vizualisation avec des fonction de rotation et de zoom. Aussi des points d'intéractions qui une fois cliqué faisait apparaître différentes information sur le rocher. Il est aussi possible de visualiser le rocher de nuit. Une partie de la ville a était recréée de manière abstraite pour pouvoir mieux intégrer le modèle sans pour autant le masquer.

Ce projet est terminé




Un autre modèle était aussi fournit et l'entreprise voulais avoir une scène de cette dernière, il est possible d'y accéder via un menu déroulant à gauche
